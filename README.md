# 8 Puzzle Solver using AI Agent

This project solves the 8-puzzle problem using the A* algorithm with Manhattan distance heuristic and visualizes the solution using a GUI.

## Features
- A* Search Algorithm
- Manhattan Distance Heuristic
- AI Agent (Perceive → Think → Act)
- Animated visualization using Tkinter

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run the main file:

bash
python "8 puzzle solver with AI Agent animation.py"
